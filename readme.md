## Problem Statement
Students face high levels of stress, anxiety, and loneliness but often hesitate to approach professional counselors.  

## Solution
This project is a safe chatbot that:
- Detects user mood using **sentiment analysis** (Very Positive, Positive, Neutral, Negative, Very Negative)
- Provides **empathetic and motivational responses**
- Tracks **mood trends** in an interactive chart
- Offers **relaxation tips and mental health resources**
- Ensures **privacy** â€” no permanent storage of messages

## Features
- Chat interface with interactive user/bot bubbles
- Mood tracking chart for session analysis
- Sidebar with helplines and mental health resources
- Clear Chat option for new sessions
- Dark theme UI for comfortable reading
